Sitio estático optimizado para GitHub Pages, con foco en postulación docente, innovación educativa, STEM, XR, IoT, inclusión y evidencias de impacto.

## Mejoras aplicadas

- Narrativa profesional alineada a propósito, impacto, altas expectativas y ciudadanía global.
- Nueva sección específica para Global Teacher Prize Chile 2026.
- Proyectos destacados organizados por valor pedagógico y evidencia.

- Bio imprimible en `assets/cv-francisco-pinto.html`.
- Viewers preparados para 3D, AR y VR.

## Cómo publicarlo en GitHub Pages

1. Descomprime este paquete.
2. Copia todos los archivos sobre tu repositorio `portafolio-profesional`.
3. Confirma que `index.html` quede en la raíz del repositorio.
4. Haz commit y push a GitHub.
5. Espera la actualización de GitHub Pages.
6. Abre el sitio y presiona `Ctrl + F5` para limpiar caché del navegador.

## Rutas principales

- `/index.html` — portada profesional.
- `/assets/cv-francisco-pinto.html` — bio imprimible.
- `/viewer3d/` — espacio para modelos 3D.
- `/viewerAR/` — espacio para realidad aumentada.
- `/viewerVR/` — espacio para realidad virtual.
- `/assets/models/` — carpeta para archivos `.glb`, `.usdz` o recursos 360.
